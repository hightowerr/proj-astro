## Description
Chekhov's Gun is a principle stating that everything that appears in a story must serve some overall purpose in the wider plot. The playwright Anton Chekhov advised that a gun hanging on the wall in one scene must go off later in the text. Otherwise, its presence is unnecessary. This principle emphasizes that every element should contribute to the overall narrative and that extraneous details should be eliminated. The principle promotes economy of storytelling and ensures that audiences aren't distracted by irrelevant elements.

## When to Avoid (or Use with Caution)
- When you're writing experimental or stream-of-consciousness narratives
- In situations where the goal is to create atmosphere rather than a tight plot
- When you want to deliberately mislead or create red herrings
- In reality-based narratives where not everything needs a narrative purpose
- When establishing mood or ambience is more important than plot progression
- In abstract or artistic works where meaning is not linear

## Thinking Steps
1. Identify all the elements currently in your narrative or presentation
2. Assess the purpose each element serves in the overall story or argument
3. Determine if each element contributes to the main theme or plot
4. Consider whether removing an element would weaken the overall work
5. Evaluate if any elements are merely decorative without functional purpose
6. Remove or modify elements that don't serve the central purpose
7. Ensure that setup elements lead to payoffs later in the work
8. Verify that everything serves the intended audience and purpose

## Coaching Questions
- What is the central purpose or theme of this work?
- Does each element contribute to that central purpose?
- What would happen if I removed this specific element?
- How does this element connect to the broader narrative or argument?
- Is this element setup for something that pays off later?
- Am I including this element just because it's interesting or clever?
- Does this element advance the story, develop character, or enhance theme?
- What would I gain by removing elements that don't serve the purpose?
- How can I ensure everything I include has a meaningful function?
