# Feedback Loops

**Description:**
Feedback Loops occur when outputs of a system are routed back as inputs, creating a chain of cause-and-effect that forms a circuit. Positive feedback amplifies changes and can lead to exponential growth or collapse, while negative feedback dampens changes and promotes stability. These loops are fundamental to understanding how systems regulate themselves and respond to changes. They can be found in biological, economic, social, and technological systems.

**When to Avoid (or Use with Caution):**
- When not identifying existing feedback loops that affect outcomes
- When designing systems without considering how they will regulate themselves
- When ignoring the amplifying effects of positive feedback loops
- When not recognizing how small changes can create large effects through feedback
- When failing to consider the time delays inherent in feedback systems

**Keywords for Situations:**
Systems design, regulation, growth, stability, dynamics, control, adaptation, evolution, business cycles, economic policy.

**Thinking Steps:**
1. **Identify Feedback Loops:** Map how outputs from the system feed back as inputs.
2. **Determine Loop Type:** Identify whether loops amplify (positive) or dampen (negative) changes.
3. **Consider Time Delays:** Understand how delays in feedback affect system behavior.
4. **Predict Behaviors:** Anticipate whether the system will stabilize, oscillate, or grow exponentially.
5. **Design Interventions:** Consider how to modify feedback loops to achieve desired outcomes.
6. **Monitor Dynamics:** Track how the system's behavior changes over time.

**Coaching Questions:**
* "What feedback loops currently exist in this system?"
* "Are these loops positive (amplifying) or negative (stabilizing)?"
* "How long does it take for feedback to affect the system?"
* "What would happen if I strengthened or weakened certain feedback loops?"
* "Am I accounting for potential time delays in system responses?"
* "How might I design beneficial feedback to create the outcomes I want?"
