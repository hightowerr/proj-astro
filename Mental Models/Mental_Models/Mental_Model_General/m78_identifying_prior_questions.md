
# Identifying Prior Questions

**Description**: A technique for deconstructing a complex question by first identifying and answering the simpler, prerequisite questions it presupposes. Many complex questions cannot be answered directly until a series of "prior" questions are resolved. This method helps to clarify the logic, assumptions, and components embedded in a complex issue before attempting to answer it directly. It is a way of disciplining one's thinking to ensure a logical and thorough exploration.

**When to Avoid (or Use with Caution)**: This technique can lead to an infinite regress if not managed with discipline. The goal is to identify the *most relevant* prior questions that make the complex question tractable, not every single question that could possibly be asked. It is most useful for deep, conceptual, or highly complex questions (e.g., "What is history?") and may be overkill for more straightforward, practical problems.

**Keywords for Situations**: Deconstruction, problem definition, research, deep thinking, clarifying complex issues, philosophical questions, systems analysis.

**Thinking Steps**:
1.  Start with the main, complex question you are focused on (e.g., "What is multiculturalism?").
2.  Formulate as many questions as you can think of that you would have to answer *before* you could answer the main question (e.g., "What is culture?", "What factors determine a person's culture?").
3.  Take this new list of questions and, for each one, determine what *further* prior questions they presuppose. For example, before answering "What is culture?", you might need to answer, "What are the shared elements of a culture (e.g., language, values, traditions)?"
4.  Continue this process, creating a hierarchy of questions that reveals the logical structure of the main question.
5.  Use this list of simpler, prior questions to guide your research and reasoning process, starting with the most foundational questions first.

**Coaching Questions**:
*   "That's a very complex question. Before we try to answer it, what are some smaller, prior questions we would need to answer first?"
*   "What does this question take for granted? What assumptions are embedded in it that we should probably question first?"
*   "To answer 'What is X?', it seems we first need to settle the question of 'What is Y?'. Let's map out the logical chain of questions here."
*   "What's the most fundamental question we need to settle before we can make progress on this larger issue?"
