# Rebalancing

## Description
Rebalancing is a critical law of business model decomposition that states: "Create value at every point where you attempt to capture value, and capture value at every point where you attempt to create value." This principle helps companies maintain economic sustainability when implementing preemptive decoupling or adapting to market changes. The rebalancing approach ensures that businesses don't give away value without capturing it, and don't capture value without creating equivalent value for customers. It's particularly important when transitioning from coupled to decoupled business models, as companies must find new ways to capture value from activities that previously subsidized other activities.

## When to Avoid (or Use with Caution)
Rebalancing should be used with caution when customers are extremely price-sensitive and can't tolerate new value-capture points. Avoid this approach if it would significantly complicate the customer experience or confuse pricing structures. Use caution when the market is not ready to pay for value that was previously free. Also be careful not to rebalance in ways that alienate your core customer base or make your offerings less competitive. Be cautious if your systems aren't sophisticated enough to track and bill for more granular value-creation activities.

## Thinking Steps
1. Map out all activities in your customer value chain where you create value
2. Identify which value-creating activities you're not currently capturing value from
3. Determine how to capture value at each point where you create value
4. Identify activities where you capture value but may not create equivalent value
5. Consider how to create more value for activities where you heavily capture value
6. Assess the customer's perspective on value creation vs. value capture balance
7. Design pricing or service structures that align value creation and capture
8. Plan for the customer communication needed to explain rebalanced value exchange
9. Create metrics to monitor the balance between value creation and capture
10. Continuously adjust the rebalancing based on customer feedback and market response

## Coaching Questions
- Where in your current model do you create value without capturing it?
- At which points do you capture value without creating equivalent customer value?
- How would customers react to you capturing value at currently free touchpoints?
- What value do you provide that customers might be willing to pay more for?
- How could you create more value at points where you currently only capture value?
- What would happen to your customer relationships if you rebalanced your value exchange?
- How do your value creation and capture points compare to competitors?
- What new pricing models would support more balanced value exchange?
- How might you experiment with rebalancing on a small scale first?
- How would you communicate rebalancing changes to minimize customer resistance?
