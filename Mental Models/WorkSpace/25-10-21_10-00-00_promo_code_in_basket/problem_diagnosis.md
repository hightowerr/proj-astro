The user is considering adding a feature that allows customers to enter promotional codes directly within the shopping basket.

**Problem:** Currently, there is friction in the user journey for applying promo codes. A significant number of customers (34%) who copy a promo code fail to apply it in the basket.

**Proposed Solution:** Implement a text field in the basket where users can directly type or paste a promo code.

**Goals:**
*   Increase the successful application of promo codes.
*   Reduce user friction.
*   Potentially unlock future features like a quicker checkout process.

**Key Considerations:**
*   The feature will be A/B tested.
*   The user flow needs to handle both successful and failed code validations.
*   The system should support applying multiple codes in sequence.

The analysis should help in making a well-informed decision about the implementation and rollout of this feature.
