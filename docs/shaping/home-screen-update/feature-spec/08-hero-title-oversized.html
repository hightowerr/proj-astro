#8 — Hero Title Oversized

Component: atelier-dashboard.tsx · Hero Section Title

Problem

The hero title uses text-5xl md:text-7xl which renders up to 72px.

RoleDS TokenSize
Display (largest)--al-display-lg56px
Page title--al-display-md44px
Current hero—72px (overshoots)

The editorial scale tops out at display-lg (56px). 72px exceeds the maximum.

Fix

Cap the hero title at --al-display-lg (56px):

/* Before */
<h1 className="text-5xl md:text-7xl">

/* After */
<h1 style={{ fontSize: 'var(--al-display-lg)' }}>
/* or use the DS typography class */
<h1 className="al-display-lg">
