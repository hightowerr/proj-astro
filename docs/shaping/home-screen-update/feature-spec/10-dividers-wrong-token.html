#10 — Dividers Use Surface Color, Not Hairline Token

Component: atelier-dashboard.tsx · Section Dividers

Problem

Dividers use border-b border-al-surface-container-high (#e8e8e6) instead of the prescribed hairline token.

The DS specifies --al-hairline (rgba(195,198,209,.20)) for the signature 1px divider.

Fix

Switch dividers to the hairline token:

/* Before */
<div className="border-b border-al-surface-container-high">

/* After */
<div style={{ borderBottom: '1px solid var(--al-hairline)' }}>
/* or with Tailwind config mapping */
<div className="border-b border-al-hairline">
