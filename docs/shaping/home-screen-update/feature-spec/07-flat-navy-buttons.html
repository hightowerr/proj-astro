Moderate — Token / Scale Violation

#7 — Primary Buttons Are Flat Navy, Not Gradient CTA

Component: atelier-dashboard.tsx · "Connect Now" & "Open Booking Page"

Problem

"Connect Now" and "Open Booking Page" use bg-al-primary (flat #001e40).

DS rule: every primary button uses the 135deg navy gradient with the CTA shadow.

  • Current shadow: 0px_8px_24px_rgba(0,30,64,0.28)
  • DS shadow: 0 14px 28px rgba(0,30,64,.2) (--al-shadow-cta)

Fix

Apply the DS gradient and shadow tokens:

/* Before */
.btn-primary {
  background: #001e40;          /* flat */
  box-shadow: 0px 8px 24px rgba(0, 30, 64, 0.28);
}

/* After */
.btn-primary {
  background: var(--al-gradient-cta);  /* 135deg → #003366 */
  box-shadow: var(--al-shadow-cta);    /* 0 14px 28px rgba(0,30,64,.2) */
}
Better approach: Use the DS Button component directly so gradient and shadow are applied automatically.
