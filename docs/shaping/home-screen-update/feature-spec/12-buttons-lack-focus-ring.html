#12 — Buttons Lack Focus Ring

Component: atelier-dashboard.tsx · All Interactive Controls

Problem

The sidebar links have focus-visible:ring, but the dashboard's <Link> and <button> CTAs don't.

The DS recommends extending the input focus ring (--al-focus-ring) to all interactive controls for accessibility.

Fix

Add focus-visible ring to all interactive elements in the dashboard:

/* Apply to all dashboard buttons and links */
<button className="... focus-visible:ring-2
  focus-visible:ring-[var(--al-focus-ring)]
  focus-visible:ring-offset-2">

<Link className="... focus-visible:ring-2
  focus-visible:ring-[var(--al-focus-ring)]
  focus-visible:ring-offset-2">

Affected elements:

  • "Connect Now" button
  • "Open Booking Page" button
  • All card-level <Link> components
  • Copy URL button
  • Any other clickable elements in the dashboard
