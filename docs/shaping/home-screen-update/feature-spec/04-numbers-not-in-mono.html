Moderate — Token / Scale Violation

#4 — Numbers Not in Mono / Tabular-Nums

Component: atelier-dashboard.tsx · Numeric Data Elements

Problem

The following data strings are rendering in Manrope instead of JetBrains Mono:

  • +14% This Month
  • 12 Items Low
  • Step 1 of 2
  • The booking URL

The DS requires all numeric/data content to use JetBrains Mono with tabular-nums for alignment and visual consistency.

Fix

Apply font-mono tabular-nums (or the .al-num utility class) to all numeric pills and the URL string.

/* Tailwind utility approach */
<span className="font-mono tabular-nums">+14% This Month</span>
<span className="font-mono tabular-nums">12 Items Low</span>
<span className="font-mono tabular-nums">Step 1 of 2</span>

/* Or use the DS utility class */
<span className="al-num">+14% This Month</span>
