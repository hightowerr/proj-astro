Major — Breaks a Brand Law

#1 — Wrong Icon Set

Component: atelier-dashboard.tsx · Home Hub

Problem

Every dashboard icon — calendar, arrow, Package, BarChart3, Users, UserPlus, ExternalLink, Copy, ChevronRight — is imported from lucide-react.

The design system mandates Material Symbols Outlined exclusively — no second icon set is permitted. The sidebar (app-nav.tsx) already does this correctly; the dashboard doesn't, creating an inconsistency.

Fix

Swap every icon to <span className="material-symbols-outlined"> (or the DS Icon component).

Icon mapping:

  • calendar → calendar_month
  • arrow → arrow_forward
  • Package → inventory_2
  • BarChart3 → bar_chart
  • Users → group
  • UserPlus → person_add
  • ExternalLink → open_in_new
  • Copy → content_copy
  • ChevronRight → chevron_right

Use FILL 1 for active/emphasis states.

<span className="material-symbols-outlined" style={{ fontVariationSettings: "'FILL' 1" }}>
  calendar_month
</span>
