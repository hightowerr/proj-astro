Moderate — Token / Scale Violation

#3 — Off-Palette Status Pill

Component: atelier-dashboard.tsx · Sales & Growth "+14%"

Problem

The "+14%" pill in Sales & Growth uses Tailwind defaults:

  • bg-emerald-100
  • text-emerald-800

DS positive green is #0e7a55 on rgba(14,122,85,0.10).

Note: The red "12 Items Low" pill maps correctly to error-container/on-error-container — leave it. However, consider whether a status pill is the right pattern here, since it reads as a status in a context that isn't really one. A muted pill may be more appropriate.

Fix

Replace Tailwind color classes with DS status tokens:

/* Before */
.pill-positive {
  background: theme('colors.emerald.100');
  color: theme('colors.emerald.800');
}

/* After */
.pill-positive {
  background: var(--al-status-positive-bg);
  color: var(--al-status-positive);
}
