#3 — Missing AL Tokens in Production

File: src/app/globals.css · Foundation · Wave 1

Dependencies: None · Parallel: #01, #05, #07

Problem

The design system defines a full typography scale, spacing scale, and motion tokens in docs/design-system/tokens/, but most of these were never added to the production globals.css. Only --al-display-lg and --al-track-eyebrow exist; the rest are missing.

This forces pages to hardcode pixel values inline instead of referencing design tokens.

Fix

Add these tokens to the :root block in globals.css (after line 389, inside the existing AL tokens section):
