#12 — Booking Page: Convert Inline Styles to Tailwind

File: src/app/book/[slug]/page.tsx · Page Conversion · Wave 4

Dependencies: #04 — needs .al-eyebrow

Problem

The public booking page uses ~25 inline style={{}} objects. It has 4 conditional branches (serviceParam, single service, no services, multiple services) that each repeat the same header pattern with inline styles:

/* Header repeated 4x with inline styles */
style={{ fontSize: '32px', fontWeight: 800,
         letterSpacing: '-0.02em', color: 'var(--al-primary)' }}

/* Layout */
style={{ padding: '48px 64px 0' }}
style={{ padding: '24px 64px 48px', maxWidth: '624px' }}

/* Eyebrow repeated 4x */
style={{ fontSize: '11px', fontWeight: 800,
         letterSpacing: '0.2em', textTransform: 'uppercase' }}

Note: This is a customer-facing page with intentionally wider padding (64px vs 48px app standard). The max-width (624px) for the form column is also intentionally narrower.

Fix

  • Background wrapper → className="bg-al-surface min-h-screen" (note: bg-al-background has no @theme inline mapping; bg-al-surface resolves to the same #f9f9f7)
  • Header section → className="pt-12 px-16" (keep wider padding for public page)
  • Eyebrow → className="al-eyebrow mb-3"
  • Title (32px) → className="text-[2rem] font-extrabold tracking-tight text-al-primary"
  • Form container → className="px-16 pb-12 max-w-[624px]"
  • All var(--al-*) inline style references → equivalent Tailwind AL classes

Consider extracting the repeated header pattern (eyebrow + title + lede) into a local BookingHeader component to eliminate the 4x duplication.

Target: Zero style={{}} props.
