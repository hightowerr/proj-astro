#17 — Calendar Settings: Convert Mixed Styles

File: src/app/app/settings/calendar/page.tsx · Typography Alignment · Wave 4

Dependencies: #04 — needs .al-page-title, .al-eyebrow

Problem

The calendar settings page mixes Tailwind classes with inline style={{}} for colors and animation:

/* Inline background */
style={{ background: "var(--al-background)" }}

/* Inline breadcrumb color */
style={{ color: "var(--al-on-surface-variant)" }}

/* Inline animation */
style={{ animation: "fade-up 0.4s ease-out both" }}

/* Title too large */
className="text-4xl md:text-5xl font-extrabold"  /* 48px, should be 44px */

/* Breadcrumb not using eyebrow pattern */
className="text-sm font-medium"

Fix

  • Remove style={{ background: "var(--al-background)" }} → add bg-al-background class (or use al-page)
  • Remove style={{ color: "var(--al-on-surface-variant)" }} → text-al-on-surface-variant
  • Remove style={{ animation: "fade-up..." }} → animate-fade-up (already defined in @theme inline)
  • Title: text-4xl md:text-5xl → al-page-title
  • Breadcrumb: text-sm font-medium → al-eyebrow
  • Align padding to canonical py-8 px-12

Target: Zero inline style={{}} props on this page.
