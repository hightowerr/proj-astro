Moderate — Missing Shared Patterns

#4 — AL Utility Classes

File: src/app/globals.css · Foundation · Wave 3

Dependencies: #02 (implemented), #03 (implemented). Note: --al-font-mono already exists (Wave 1 Slice 1). Most #03 tokens exist, but --al-track-eyebrow (0.2em) and --al-radius-3xl (24px) must be added to production CSS first — they exist in docs/design-system/tokens/ but were never added to globals.css.

Problem

Pages repeat the same styling patterns inline (eyebrows, page titles, section titles, card containers, lede text) because no shared utility classes exist in production. The design system docs define .al-eyebrow, .al-card, .al-num, and .al-mono in docs/design-system/tokens/, but these were never added to globals.css.

Without these classes, every page re-invents the same patterns with slightly different values — causing the inconsistencies across the app.

Fix

Pre-req: Add two missing tokens to the :root block before adding utilities:

/* Add to Tracking section (~line 407) */
--al-track-eyebrow: 0.2em;

/* Add to Roundness section (~line 452) */
--al-radius-3xl:  24px;

Then add the following utility classes at the end of the "Atelier Light Utilities" section of globals.css (after the toggle switch block, approximately line 724+; exact line may vary):

/* Standard app page wrapper */
.al-page {
  font-family: var(--al-font);
  background: var(--al-background);
  min-height: 100vh;
  padding: var(--al-main-pad-y) var(--al-main-pad-x);
  display: flex;
  flex-direction: column;
  gap: var(--al-sp-6);
}

/* Page title — canonical 44px / 800 */
.al-page-title {
  font-size: var(--al-display-md);
  font-weight: var(--al-w-extra);
  letter-spacing: var(--al-track-display);
  color: var(--al-primary);
  line-height: 1.0;
}

/* Section title — 24px / 800 */
.al-section-title {
  font-size: var(--al-headline-md);
  font-weight: var(--al-w-extra);
  letter-spacing: -0.02em;
  color: var(--al-primary);
  line-height: 1.1;
}

/* Eyebrow motif — the signature element */
.al-eyebrow {
  font-family: var(--al-font);
  font-size: var(--al-label-sm);
  font-weight: var(--al-w-extra);
  letter-spacing: var(--al-track-eyebrow);
  text-transform: uppercase;
  color: var(--al-on-surface-variant);
  opacity: 0.55;
}

/* Body lede / description text */
.al-lede {
  font-size: var(--al-title-sm);
  color: var(--al-on-surface-variant);
  line-height: 1.55;
  max-width: 62ch;
}

/* Card container — the single elevation recipe */
.al-card {
  background: var(--al-surface-container-lowest);
  border-radius: var(--al-radius-3xl);
  box-shadow: var(--al-shadow-float);
  overflow: hidden;
}

/* Tabular numerics utility */
.al-num {
  font-variant-numeric: tabular-nums;
}

/* Monospace numerics (time, money, counts) */
.al-mono {
  font-family: var(--al-font-mono);
  font-variant-numeric: tabular-nums;
}

Canonical page structure:

<div className="al-page">
  <div className="al-eyebrow">Studio / Page Name</div>
  <h1 className="al-page-title">Page Title</h1>
  <p className="al-lede">Description text.</p>
  <section>
    <h2 className="al-section-title">Section</h2>
    <div className="al-card p-6">...</div>
  </section>
</div>
