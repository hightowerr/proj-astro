#18 — Profile Page: Align to Atelier Light

File: src/app/profile/page.tsx · Typography Alignment · Wave 4

Dependencies: #04 — needs .al-page, .al-page-title, .al-card

Problem

The profile page looks like a different app. It uses the default shadcn/ui Card aesthetic with no Atelier Light styling:

  • Container: container max-w-4xl mx-auto py-8 px-4 — different from canonical al-page padding
  • Title: text-3xl font-bold (30px/700) — should be al-page-title (44px/800)
  • Uses Lucide icons (Mail, Calendar, ArrowLeft, etc.) instead of Material Symbols
  • Hardcoded green colors: text-green-600, border-green-600, bg-green-500 — should use AL status tokens
  • Uses text-muted-foreground everywhere — works via shadcn compat but inconsistent with other pages using text-al-on-surface-variant
  • Card radius: default shadcn rounded-lg (8px) instead of canonical rounded-3xl (24px)

Fix

  • Adopt al-page wrapper for consistent page shell
  • Title: text-3xl font-bold → al-page-title
  • Replace Lucide icons (imported on line 5: Mail, Calendar, User, Shield, ArrowLeft, Lock, Smartphone) with Material Symbols Outlined equivalents:
    • Mail → mail
    • Calendar → calendar_month
    • ArrowLeft → arrow_back
    • User → account_circle
    • Shield → shield
    • Lock → lock
    • Smartphone → smartphone
  • Replace hardcoded green colors with AL status tokens
  • Replace text-muted-foreground with text-al-on-surface-variant
  • Apply al-card class to card containers (gets 24px radius + float shadow)
