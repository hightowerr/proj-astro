#10 — Customers Page: Convert Inline Styles to Tailwind

File: src/app/app/customers/page.tsx · Page Conversion · Wave 4

Dependencies: #02 (inline fontFamily removal assumes base font fix), #04 (utility classes)

Problem

The customers list page uses ~13 inline style={{}} objects with hardcoded colors and a redundant fontFamily declaration. Note: the child component src/components/customers/customers-editorial.tsx contains ~88 additional inline styles and should be scoped into this migration or handled as a companion spec.

/* Root container */
style={{ padding: "32px 48px", display: "flex", flexDirection: "column",
         gap: 24, fontFamily: "'Manrope', system-ui, sans-serif" }}

/* Hardcoded hex colors */
"#001e40", "#43474f"

/* Inline font-family on individual elements */
fontFamily: "'Manrope', system-ui, sans-serif"

The hardcoded fontFamily strings are redundant since the base layer already sets Manrope (after Spec #2 is applied).

Fix

  • Root wrapper → className="al-page"
  • Remove all inline fontFamily declarations (redundant after base font fix)
  • Page title → className="al-page-title"
  • Eyebrow labels → className="al-eyebrow"
  • Description text → className="al-lede"
  • Card containers → className="al-card"
  • Buttons with inline gradient → className="al-gradient-cta" + Tailwind padding/radius
  • Buttons with inline border → Tailwind border classes with AL tokens
  • Hardcoded hex colors → AL text/bg utility classes
  • Inline padding/gap → Tailwind spacing classes

Target: Zero style={{}} props. All styling via Tailwind + AL classes.
