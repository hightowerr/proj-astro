Multiple files · Typography Alignment · Wave 4

Dependencies: #04 — needs .al-page-title, .al-section-title, .al-lede

Problem

Several pages already use Tailwind classes correctly but have minor typography mismatches with the canonical design:

PageCurrent titleContainer
settings/availability/page.tsxtext-3xl font-semibold (30px/600)container mx-auto px-4 py-10
settings/reminders/page.tsxtext-lg font-semibold for sectionscontainer mx-auto px-4 py-10

chat/page.tsx (path: src/app/chat/, not src/app/app/)text-2xl font-boldcontainer mx-auto px-4 py-8 (loading states use py-12)

All page titles should be al-page-title (44px/800). Container padding should be py-8 px-12 to match --al-main-pad-*.

Fix

For each page:

  • Page title class → al-page-title
  • Subtitle / section head class → al-section-title or al-lede as appropriate
  • Container padding: px-4 py-10 → px-12 py-8
  • Replace text-muted-foreground with text-al-on-surface-variant for consistency

These are small changes per file but collectively establish a consistent feel across all detail/settings pages.
