#1 — Font Loading Cleanup

File: src/app/layout.tsx · Foundation · Wave 1

Dependencies: None · Parallel: #03, #05, #07

Problem

The root layout loads four Google Fonts via next/font/google, but the Atelier Light design system only permits three font families:

Currently loadedDesign system status
Cormorant_Garamond (--font-cormorant)Not in design system. Forbidden.
Bricolage_Grotesque (--font-bricolage)Not in design system. Forbidden.
Fira_Code (--font-fira-code)Design system specifies JetBrains Mono instead.
Manrope (--font-manrope-raw)Correct. Keep.

Additionally, JetBrains Mono (required for all numeric data: time, money, counts) is never loaded. Numerics currently fall back to OS monospace.

Fix

  1. Remove the Cormorant_Garamond, Bricolage_Grotesque, and Fira_Code imports and their const declarations.
  2. Add JetBrains_Mono import: 
    import { Manrope, JetBrains_Mono } from "next/font/google";

const jetbrainsMono = JetBrains_Mono({
  subsets: ["latin"],
  weight: ["400", "500", "600", "700"],
  variable: "--font-jetbrains-mono",
  display: "swap",
});
  3. Update the body className: 
    className={`${manrope.variable} ${jetbrainsMono.variable} antialiased bg-background`}
  4. Keep the Material Symbols Outlined <link> tag unchanged (it's correct).
