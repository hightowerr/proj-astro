Feature Spec

Service Card

Displays the currently selected service with its name, duration, and timezone. Shows a "Selected" status badge. When multiple services are available, this acts as the selectable service option.

Design notes

  • Card background: #f4f4f2 (surface-container-low), radius 16px
  • Eyebrow: 10px, weight 800, 0.2em tracking, uppercase, muted @ 0.55
  • Service name: 16px, weight 700, navy (#001e40)
  • Meta line: 13px, muted (#43474f), format "[duration] · [timezone]"
  • Badge: green pill (#0e7a55 text on rgba(14,122,85,0.10) bg), radius 9999px, 11px weight 700, 0.16em tracking
  • Badge includes a 6px green dot before the text
  • Padding: 20px vertical, 24px horizontal
  • When only one service exists, it's pre-selected and non-interactive
  • Design change only — no new functionality
