Design notes

  • Eyebrow: 11px, weight 800, 0.2em tracking, uppercase, #43474f @ 0.55 opacity
  • Heading: 32px (headline-lg), weight 800, -0.02em tracking, navy (#001e40)
  • Subtitle: 16px (body-md), weight 400, muted (#43474f), shows "[Service] · [Duration]"
  • Spacing: 12px below eyebrow, 8px below heading
  • Content area: padded 48px top, 64px horizontal — centered on the page background
  • The shop name ("[slug]") is dynamic, pulled from the booking link slug
  • Design change only — no new functionality
