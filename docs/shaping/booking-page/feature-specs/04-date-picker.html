Feature Spec

Date Picker

A date input field that lets the customer select a booking date. Uses a native date input styled to match the Atelier Light theme. The required indicator is a small navy dot next to the label.



Design notes

  • Label: 13px, weight 800, navy (#001e40)
  • Required indicator: 6px navy dot (not an asterisk)
  • Input wrap: white bg, 1px border rgba(195,198,209,0.50), radius 12px, 14px 16px padding
  • Input text: 16px, weight 700, navy (#001e40)
  • Calendar icon: 20px Material Symbol, muted (#737780), right-aligned
  • Focus state: navy border + focus ring 0 0 0 3px rgba(0,30,64,0.12)
  • Uses native <input type="date"> — browser date picker pops up on click
  • Default value: next available booking date
  • Design change only — no new functionality
