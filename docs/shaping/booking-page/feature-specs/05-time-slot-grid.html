Feature Spec

Time Slot Grid

A grid of available time slots for the selected date. Each slot is a selectable button showing the start time. Only one slot can be selected at a time.

Available slots
60 minutes · UTC

Design notes

  • Label: 13px, weight 800, navy with 6px navy dot required indicator
  • Meta line: 13px, muted (#43474f), shows "[duration] · [timezone]"
  • Grid: flex-wrap with 10px gap
  • Slot button default: white bg, 1px border rgba(195,198,209,0.50), radius 12px, 12px 20px padding
  • Slot button text: 13px monospace (JetBrains Mono), weight 600, tabular-nums, muted (#43474f)
  • Slot hover: #eeeeec bg, lighter border
  • Slot selected: navy bg (#001e40), white text, navy border, weight 700, ring shadow 0 0 0 4px rgba(0,30,64,0.08)
  • Single selection — clicking a new slot deselects the previous one
  • Slots are dynamically generated based on shop availability for the chosen date
  • Design change only — no new functionality
