Design notes

  • Brand mark: 32px navy square, radius 8px, Material Symbol "dashboard_customize" centered in white
  • Brand name: 18px, weight 800, 0.04em tracking, uppercase, navy
  • Nav links: 14px, weight 600, muted ink (#43474f), 10px radius hover bg (#eeeeec)
  • Sign in: plain text link, same style as nav links
  • CTA button: navy gradient (135deg #001e40 → #003366), white text, 13px weight 700, radius 12px, shadow 0 14px 28px rgba(0,30,64,.2)
  • Bottom hairline: 1px solid rgba(195,198,209, 0.20)
  • No sticky/fixed positioning — scrolls with page
  • Design change only — no new functionality
