Feature Spec

Communication Preferences

Two opt-in controls for booking notifications. SMS updates is a simple inline checkbox (unchecked by default). Email reminders is a card-style checkbox with a description (checked by default).

Design notes

  • Two distinct checkbox styles for visual hierarchy
  • SMS opt-in: inline layout, unchecked by default, plain text label 14px weight 500
  • Email reminders: card layout with #f4f4f2 bg, radius 12px, 16px 20px padding, checked by default
  • Checkbox: 20px square, 2px border, radius 6px, white bg unchecked
  • Checkbox checked: navy bg (#001e40), navy border, white checkmark
  • Email card has title (14px, weight 700) and description (13px, muted, line-height 1.5)
  • Description mentions "You can opt out later" — sets expectation for manage page toggle
  • 12px gap between the two preference items
  • Design change only — no new functionality
