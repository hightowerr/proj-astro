Feature Spec

Contact Form

Customer contact information fields: full name (full width), then phone and email side-by-side. All fields are required. Placeholders provide example formatting.

Design notes

  • Full name: full-width row
  • Phone + Email: 2-column grid with 16px gap
  • Labels: 13px, weight 800, navy (#001e40), with 6px navy dot required indicator
  • Input wrap: white bg, 1px border rgba(195,198,209,0.50), radius 12px, 14px 16px padding
  • Input text: 16px, weight 700, navy (#001e40)
  • Placeholder: 16px, weight 400, muted (#737780)
  • Focus state: navy border + focus ring 0 0 0 3px rgba(0,30,64,0.12)
  • 20px bottom margin between field groups
  • All three fields are required — validated on form submission
  • Design change only — no new functionality
