# Honest positioning at low adoption

ShowUp's landing page uses future social proof (Cialdini) instead of fabricated user counts. At n=1 customer, we removed "Trusted by 500+ beauty professionals" and replaced it with honest trend-signalling ("Early adopters are filling their calendars") rather than manufacturing credibility.

## Why

The target market (independent hairdressers, barbers, beauticians) forms a small-world network — high gossip, low anonymity. A falsifiable user count ("500+") is verifiable in one peer conversation ("Have you heard of ShowUp?" / "No"). Discovery probability approaches 100%. In this network topology, a trust violation propagates as negative signal exponentially faster than genuine advocacy. The cost of being caught lying about user count is permanent — the cost of being honest about being new is temporary (resolved by acquiring real users).

Cialdini (Influence Ch.4) identifies "future social proof" as equally effective to majority-behavior claims when current adoption is low: communicating an honest upward trend triggers forward projection in the reader. This is our validated alternative.

## Considered Options

| Option | Rejected because |
|---|---|
| Fabricated user count ("500+") | Falsifiable in one conversation in a tight professional network. Trust violation is irreversible. |
| Named peer reference ("See how Kicksnare...") | Requires customer consent. Reveals exactly n=1. Upgrade path when testimonial is obtained. |
| Authority borrowing ("Powered by Stripe") | Different mechanism (authority, not social proof). Weaker in the social proof slot. |
| Friction reduction ("14-day free trial") | Wastes the highest-leverage conversion position for a secondary message. Move this near the CTA button instead. |
| No social proof line at all | Leaves the strongest persuasion slot empty. |

## Consequences

- Any future copy that claims a user count MUST be backed by real data. No aspirational numbers.
- Social proof upgrades to named testimonials when available (target: 3-5 customers with consent).
- The "14-day free trial" message lives near CTA buttons, not in the social proof slot.
- Feature claims on the landing page must describe shipped capabilities only. Aspirational features require explicit "coming soon" labeling or exclusion.
