Pill-shaped label for counts, categories, and metadata; `curator` is the signature terracotta category chip.

```jsx
<Badge variant="curator">Balayage</Badge>
<Badge variant="muted">12</Badge>
<Badge variant="primary">Pro</Badge>
<Badge variant="outline">Draft</Badge>
```

- `variant` — `muted` (default neutral), `curator` (terracotta-fixed chip), `secondary` (terracotta-container), `primary` (solid navy), `outline` (hairline).
- Ration terracotta: at most ~one terracotta badge per viewport.
- For appointment/customer/payment status, use **StatusPill** instead — it carries the dot + uppercase semantics.
