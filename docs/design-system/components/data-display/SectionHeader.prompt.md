Eyebrow + title + lede stack that opens every sheet and page section.

```jsx
<SectionHeader
  eyebrow="The ledger"
  title="Recent appointments"
  lede="Settled, pending, and voided outcomes from the last 7 days."
/>
```

- `eyebrow` — uppercase 10px/800/.2em motif; keep it short (1–3 words).
- `title` — 24px/800/navy section title.
- `as` — pick the right heading level for document structure.
- The eyebrow styling is fixed across the system — never restyle it.
